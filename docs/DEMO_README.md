# Demo GIF

To create a demo GIF for the README:

1. Start Navi: `navi start`
2. Use a screen recording tool (e.g., CleanShot X, Kap, or macOS built-in)
3. Record yourself:
   - Pressing ⌘⇧N to start recording
   - Speaking a short phrase
   - Pressing ⌘⇧N to stop
   - Showing the note appear in Obsidian
4. Convert to GIF and save as `demo.gif` in this folder

Recommended dimensions: 800x500px
Recommended duration: 10-15 seconds
